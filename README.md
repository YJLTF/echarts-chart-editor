# Chart Studio - ECharts 图表编辑器

一个基于 ECharts 的可视化图表编辑器，支持多种图表类型的实时预览和配置。

## 功能特性

- **多种图表类型**：折线图、柱状图、饼图、散点图、雷达图、仪表盘、漏斗图、象形柱图等
- **实时预览**：修改配置即时查看效果
- **双主题切换**：支持浅色/深色主题
- **数据编辑**：内置数据编辑器，支持增删改数据项
- **配置导入导出**：JSON 格式配置文件的导入和导出
- **图片下载**：将图表导出为 PNG 图片
- **代码预览**：实时查看 ECharts 配置代码并复制
- **缩放控制**：支持图表缩放和全屏模式

## 项目结构

```
├── index.html          # 主页面
├── app.js              # 核心逻辑
├── style.css           # 样式文件
└── lib/
    └── echarts.min.js  # ECharts 库（本地依赖）
```

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需构建步骤。

## 技术栈

- 原生 HTML/CSS/JavaScript
- [ECharts 5.5.0](https://echarts.apache.org/) - 图表库（本地依赖）
